Keyword Cipher

Keyword cipher is a form of monoalphabetic substitution. In this section you will be creating a prgram that will use user created keywords to encode and decode user input. The keyword provided by the user will be placed at the beginning of the alphabet proceded by the remaining letters (minus those used in keyword). Your task is to create a Keyword cipher that will take in keyword which will be used to cipher strings of information from the user and output the ciphered equivalent.

Tasks

  • Ensure the keywords are valid before setting them. Valid key words have all unique characters
  • If the Key is NOT valid, display alert message to user
  • Translator should be able to convert from Plain text to Ecrypted Phrase
  • Translator should be able to convert from Encrypted Phrase to Plain text

    • Additional tasks

  • Create unique styles for your HTML page
  • Create test cases for your code (in src/text/java/com/fdmgroup/icode/cipher)
  • Have your translator read the encrypted phrases

Grid Code

Grid code is a way of franctionating plain text into a smaller set of symbols. There are two methods you can take.
The first method is the Caesar cipher which locates the letter on the grid and returns the next letter shifted down.

The second method is the Bifid cipher which locates the letter on the grid and returns its coordinates.
Your task is to create a Grid Code tanslator that will take in string of information from the user and output the code equivalent.

Tasks

  • Translator should be able to convert from English to Grid Code (method of choice)
  • Translator should be able to convert from Grid Code (method of choice) to English

    • Additional tasks

  • Translator should be able to convert from English to Grid Code (other method)
  • Translator should be able to convert from Grid Code (other method) to English
  • Create unique styles for your HTML page
  • Create test cases for your code (in src/text/java/com/fdmgroup/icode/cipher)
