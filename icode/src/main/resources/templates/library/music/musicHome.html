Music Abstract Class


The Java abstract class "Music" represents a music entity with properties like musicId, title, artist, and genre. It serves as a base class for specific types of music such as albums and records. It includes constructors for creating instances with unique IDs, getters and setters for the properties, and a method to generate unique IDs incrementally. The toString method is overridden to provide a string representation of the Music object. The abstract method "play" is defined to be implemented by subclasses.


Album Class

Record Class
