Overview

Welcome to the Library Music Feature! This feature allows you to manage the music in our library. Usually, you can view all music, add a new album, add a new record, search for music by ID, title, artist, or genre, edit an album's description, edit a record's description, delete a record, and delete an album. But fortunately you are here now to help us fix some bugs and add some new features.
