Once in a blue moon the stars align and we can open the doors to a new location. It is important that the add new location feature works correctly to keep our community inform of where they can find us.


To implement the "Add New Location" feature, you will need to perform the following steps:

  1. Create a new controller method in your Spring application to handle the add request.
  2. Add a new button or link in the HTML code to trigger the add request.
  3. Pass the location information as parameters in the add request.
  4. In the controller method, retrieve the location information and use it to add the new location to the database.
  5. Handle any error scenarios, such as when the location information is incomplete or invalid.

Task 5


Our fifth task is to add a new feature by allowing users to search for locations by the location name. You need to add a new page that allows users to search for locations by name. Once the user enters the name, the page should display the location details.


To implement the "Search Locations by Name" feature, you will need to perform the following steps:

  1. Create a new controller method in your Spring application to handle the search request.
  2. Add a new input field and button in the HTML code to allow users to enter the location name and trigger the search request.
  3. Pass the location name as a parameter in the search request.
  4. In the controller method, retrieve the location name and use it to search for locations in the database.
  5. Display the location details on the page based on the search results.
  6. Handle any error scenarios, such as when no locations are found for the given name.
