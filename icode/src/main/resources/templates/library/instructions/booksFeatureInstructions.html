Overview


Welcome to the Library Book Feature! This feature allows you to manage the books in our library. Usually, you can view all books, add a new book, search for books by ID, title, or author, edit a book's description, and delete a book. But fortunately you are here now to help us fix some bugs and add some new features.

Task 1 Instructions:


The first task is to fix the bug in the "All Books" page. The bug is that the "All Books" page is not displaying the books. You need to fix the bug so that the "All Books" page displays the books.To fix the bug, you need to go to the "All Books" page and check the code to see what is causing the bug. Once you find the bug, you need to fix it and test the "All Books" page to make sure it is displaying the books. Once you have fixed the bug, you can move on to the next task....Good Luck!

Task 7


It is unfortunate but sometimes a book needs to be retired from the catalog for an array of reasons.


With a heavy heart, the seventh task is to delete a book by removing it from the library catalog permanently. To do this, you need to add a new page that allows users to search for books by ID and delete the book. Please note that once a book is deleted, it cannot be recovered. So, it is important to be sure that the book should be deleted by validating the user's action. Once the user enters the ID, the book should be deleted from the library catalog and the user redirected to the "All Books" page.
