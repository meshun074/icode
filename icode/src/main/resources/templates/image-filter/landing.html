Grayscale Conversion

Images are a collection of pixels and their appearance can be changed by maniplating the RGB values which determine how their color is displayed. You are on the team that will create the algorithm which will allow images to be converted to grayscale (see example above).

To achieve this goal, you will be working with BufferedImage (a Java data type) which will allow you to loop through the pixels and manipulate the values. Here is a helpful link to explore how to modify pixel values. ALL code that you complete will be in the icode/image/grayscale package.

To upload images, set a path for your upload folder in application.properties. It must be a fullyqualified path on your machine to an existing folder. Photos must be 1MB or less or they will not be uploaded. Sample images (flower and flower-field can be found in src/main/resources/static/image-filter/images)

Tasks

  • Complete Grayscale filter (ImageFilter)
    • How to achieve grayscale?
    • replace each pixel with the average of the RGB values Avg = (R+G+B)/3
  • Choose from other filters on page (can read about them in their sections) and add them to your Image Filter
  • Improve the HTML and CSS of your view-image template
  • Implement a delete functionality to delete images from storage

Tasks

  • Complete Sepia filter (ImageFilter)
    • How to achieve sepia?
    • First, create new RGB values
      • newRed = 0.393*R + 0.769*G + 0.189*B
      • newGreen = 0.349*R + 0.686*G + 0.168*B
      • newBlue = 0.272*R + 0.534*G + 0.131*B
    • Conditionally set the new pixel values
      • if newColor > 255 then C = 255 else C = newColor
