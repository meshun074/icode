Metrics Converter

Welcome to the Metrics Converter application. In order to complete this application, you will build implementation logic within methods. You will need to make use of conditional statements and getters and setters of the ResultDto object (found in icode/metrics/ResultDto).

Tasks

  • Build the implementation logic within the MetricsService class for the existing methods
  • Create new methods in the service for additional conversions listed in the options of the converter.html page.
  • Allow the Controller to use those newly created methods by adding conditional logic to the existing MetricsController method
  • Add additional conversion options that are not preset, feel free to be creative (e.g., human weight to bananas, height to pencils, etc.)

    • Additional tasks

  • Create unique styles for your HTML page
  • Create test cases for your code (in scr/test/java/com/fdmgroup/icode/metrics)
