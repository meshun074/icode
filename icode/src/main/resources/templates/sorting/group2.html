Selection Sort



This algorithm continuously selects the smallest elements from the unsorted portion of the array and places it at the end of the sorted portion.

For instance, above we start with an unsorted array. We place one pointer at the beginning of the array and have a second loop through the array, checking for the smallest element in the array. After completing the loop, 1 is identified as the smallest element. The smallest element is placed in its sorted position.

After the first elements is verified to be the smallest, the first pointer is moved and the processes is repeated until the array is sorted.

Insertion Sort



This algorithm loops through elements in an array and compares adjacent elements, moving the current (key) element to the front (beginning of the array) so long as it is smaller.

This sorting method iterates over an array and compares the current element (key) to its predecessor, if the key element is smaller than its predecessor, the elements before, then move the greater elements one position up to make space for the swapped element.

In the above example, 11 is the key element and is smaller than 12 so the two elements are swapped.

The process of comparing the key to the remainder of the elements continues until all elements are sorted (reach the end of the array).

Merge Sort


This algorithm works by dividing an array into smaller subarrays, sorting each subarray, and then merging the sorted subarrays back together to form the final sorted array. In other words, merge sort works by dividing the array into two halves, sorting each half, and then merging the sorted halves back together. This process is repeated until the entire array is sorted.

For instance, in the above example, we find the middle of the intial array (position 2 - 70). Then we sort the right and left halves (divide them further). This is achieved through the use of recursion -- a method calling itself. We continue dividing (sorting) the array so long as the left half is smaller than the right. Once the array can no longer be sorted (divided) then we start to merge the two halves together.
