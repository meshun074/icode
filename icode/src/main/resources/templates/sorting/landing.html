All Sorts of Shenanigans

Welcome to the Sorting Center where we will complete all kinds of sorting shenanigans. This application times how long it takes different sets of algorithms to sort a set of prepopulated data. In completing this application, you will gain experience problem solving in java, creating methods, working with arrays, comparators, and potentially writing your own method to populate a random data set for the algorithms to sort.

Group 1

Tasks

  • Solve each of the assigned sorting algorithms
  • Complete the comparator to list the algorithms from fastest to slowest
  • Complete the comparator to list the algorithms from slowest to fastest

    • Additional tasks

  • Create a method to produce sample data to sort (in src/main/java/com/fdmgroup/icode/sorting/data/DataToSort)
  • Create unique styles for your HTML page
  • Create test cases for your code (in src/test/java/com/fdmgroup/icode/sorting)

Start!
